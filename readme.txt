# Información del proyecto.

La idea de este proyecto es hacer una revisión del sitio web de la empresa en la que trabajo (y me pagó el curso).

# Notas para la tercer pre-entrega:

## sass

En base/utils.scss se definen varios mixins y use un map para crear clases un un bucle para volver invisibles según el tamaño de la pantalla a los elementos que usen esas clases.
En titles.scss usé extend. También para los containers.

## Utilización de frameworks

Utilicé bootstrap para los botones de navegación y el formulario de contacto.
Para las animaciones de entrada usé la librería AOS.
